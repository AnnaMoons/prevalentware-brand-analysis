# SB 4. Lista Negra de Pérdidas Innegociables - PrevalentWare

## Lo que NUNCA sacrificaremos

### Lista Negra (No Negociables):

**1. Calidad Técnica**
- Jamás entregar código sin testing
- Jamás comprometer arquitectura por velocidad
- Jamás ignorar deuda técnica

**2. Seguridad**
- Jamás exponer datos de cliente
- Jamás ignorar vulnerabilidades
- Jamás omitir controles de seguridad

**3. Integridad**
- Jamás prometer lo que no podemos cumplir
- Jamás ocultar problemas
- Jamás competir deslealmente

**4. Relación con Cliente**
- Jamás tratar al cliente como número
- Jamás ignorar feedback
- Jamás faltar al compromiso

**5. Cultura**
- Jamás contratar contra valores
- Jamás tolerar falta de ética
- Jamás sacrificar equipo por proyecto

**6. Reputación**
- Jamás aceptar proyecto que dañe marca
- Jamás asociarnos con clientes no éticos
- Jamás quemar puentes
