# MT 28. The Burn Pyramid - PrevalentWare

## Piramide de Prioridades

### Nivel 1: NO QUEMAR (Innegociable)
**Lo que jamas comprometeremos:**
- Calidad tecnica
- Seguridad de datos
- Compromiso con cliente
- Integridad

### Nivel 2: MANTENER (Importante)
**Lo que mantenemos fuerte:**
- Comunicacion frecuente
- Documentacion
- Capacitacion
- Soporte

### Nivel 3: OPTIMIZAR (Mejora continua)
**Lo que siempre mejoramos:**
- Procesos internos
- Eficiencia
- Herramientas
- Metodologias

### Nivel 4: EXPERIMENTAR (Innovacion)
**Donde innovamos:**
- Nuevas tecnologias
- POCs
- Metodologias emergentes
- Mercados nuevos

### Nivel 5: SOLTAR (Deprecar)
**Lo que dejamos ir:**
- Tecnologias obsoletas
- Procesos ineficientes
- Servicios que no diferencian
- Mercados que no ajustan
